Описание:
Мы видели breakв операторе switch, что он может выпрыгивать из оператора switch.
Его также можно использовать в операторе loop.

Пример:


function findFirstOddNumber(arr){
var result;
for (var i=0;i if (arr[i]%2==1){
result=arr[i];
break;
}
}
return result;
}
В этом примере цикл for проходит по массиву arr.
Когда встречается первое нечетное число,
breakоператор выводит оператор из цикла
for.
В операторе цикла мы можем использовать continueпропуск части кода.
Например, следующий код игнорирует отрицательные значения:

function dontDisplayNegative(arr){
for (var i=0;i if (arr[i]<0) continue; > console.log(arr[i])
}
}
В этом примере console.log(arr[i])никогда не выполняется для отрицательного элемента.
благодаря continue.

Используя breakand continue,
вы можете сделать оператор цикла более гибким и удобным.
