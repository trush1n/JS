Цикл for— очень часто используемый цикл в JS.

Помните, на уроке 9 мы написали whileцикл:

function sum1_100(){
var sum=0,num=1
while (num<=100){
sum+=num;
num++;
}
return sum;
}
Мы можем переписать это, используя forцикл, например так:

function sum1_100(){
for (var sum=0,num=1;num<=100;num++){
sum+=num;
}
return sum;
}
Как вы видите, в скобках три части, разделенные " ;".
Первая часть, var sum=0,num=1, — это инициализация. Эта часть будет запущена
до начала цикла. Вторая часть, num<=100— это условное выражение. Условное выражение проверяется перед началом каждого прохода по
циклу. Когда значение выражения ложно, цикл будет завершен.
Третья часть, num++будет запущена после блока кода и обычно
используется для увеличения и уменьшения переменных.

Для приведенного выше примера порядок выполнения кода следующий:

<--- back to part 2
-------------------------------------------
| true |
var sum=0,num=1 ---> num<=100 ? ------> code block---> num++
|
false|
loop terminated
Когда вам нужно обойти массив, вы можете использовать forцикл для доступа к каждому элементу, используя его индекс. Вот пример:

function displayElements(array){
for (var i=0;i console.log(array[i]);
}
}
Эта функция отобразит каждый элемент массива на экране.
