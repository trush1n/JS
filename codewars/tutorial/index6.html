В javascript, логический объект имеет только два значения,
trueили false. Используйте ===, ==, >, <, >=, <=между двумя переменными
(или константами), результатом такого выражения будет логическое значение. Операцию между двумя логическими
значениями можно использовать с логическими операторами: &&, ||, !или некоторыми побитовыми операторами:&, |, ^
Многие данные, вероятно, будут неявно преобразованы в булевы значения в данном контексте,
что наиболее распространено в условных операторах. Другими словами, булево значение в основном используется для условного суждения.
Посмотрите на этот пример:

var arr=[];
...
...
......
if (arr.length) return "arr has elements";
else return "arr is empty";
Как видите, arr определен как массив, arr.length должен быть числовым значением, но в этом примере числовое значение должно быть
преобразовано в логическое значение. Это означает, что если значение arr.length равно 0, преобразовать значение в false,
если значение arr.length не равно 0, преобразовать значение в true.

Эти значения преобразуются falseв условном операторе:

0, -0, "", null, undefined, NaN
