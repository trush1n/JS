Описание:

В JavaScript switchможет заменять несколько ifоператоров.


switch(n){
case 1:
//code block
break;
case 2:
//code block
break;
case ...:
//code block
break;

default: //default is optional, sometimes it can be omitted
//code block
//The last one does not need break
}

switch является ключевым словом и nявляется переменной для переключения. case 1означает когда n===1.
После " :" вы можете добавить свой код для того, что делать в этом случае.
Ключевое слово breakиспользуется для завершения - если вы забыли break,
код продолжит выполняться через другие caseоператоры и defaultдо тех пор, пока breakне появится .
Если ни один caseоператор не соответствует, defaultвводится .

В некоторых случаях switchутверждение яснее, чем if..elseутверждение.
