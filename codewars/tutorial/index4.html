В javascript Array — один из базовых типов данных.
Чтобы определить пустой массив, можно использовать var arr=new Array()илиvar arr=[]
Массив имеет атрибут: length, если существует массив с именем arr,
используется arr.lengthдля определения количества элементов, содержащихся в массиве.
Каждый элемент массива имеет index, используемый arr[index]для получения значения элемента.
indexвсегда начинаются с 0, поэтому первый элемент массива — arr[0],
последний элемент массива — arr[arr.length-1].
Если мы хотим добавить новые элементы в массив, можно использовать метод array: push().
Он может добавить элемент в конец массива. Вместо этого, если мы хотим удалить последний элемент массива,
можно использовать метод array: pop(). например:
var arr=[1,2,3]; //define an array arr contains elements 1 2 3
arr.push(4); //add element 4 to arr
console.log(arr) //[1,2,3,4]
arr.pop(); //remove the last element from arr
console.log(arr) //[1,2,3]
