Описание:

Число — это базовый тип данных в javascript. javascript также поддерживает Number objects. Объект — это исходное значение объекта пакета. При необходимости JavaScript автоматически преобразует исходные данные в объекты. Вы можете явно создать объект Number с помощью конструктора Number(). Хотя это не обязательно. Использование: var num=new Number(value);

Параметр value— это значение объекта Number, которое необходимо создать, или значение, которое необходимо преобразовать в числовое значение.

Конструктор Number()может использоваться без оператора newи напрямую как функция преобразования для использования. Таким образом, когда вызывается Number, он преобразуется в число, а затем возвращает преобразованное значение (или NaN). Использование:

var num=Number(value); Объект Number имеет два общих свойства объекта: constructorи prototype. Все объекты в JS имеют эти два свойства. Это два очень важных атрибута. Из-за их важности и сложности мы изучим их в будущем.

Помимо двух вышеупомянутых, объекты Number имеют пять атрибутов (или называемых константами):

MAX_VALUE: Максимальное число, которое может быть выражено в JS. Использование: Number.MAX_VALUE. Его приблизительное значение равно1.7976931348623157e+308

MIN_VALUE: Минимальное число, которое может быть выражено в JS (близко к 0, но не отрицательно). Использование: Number.MIN_VALUE. Его приблизительное значение равно5e-324

NaN: Нечисловое значение. Сокращения от "Not a Number". Когда некоторые арифметические операции (например, квадратный корень отрицательного числа) или результат метода не являются числами, возвращается NaN. Использование: Number.NaN. Его можно упростить и заменить на NaN.

Обратите внимание: результаты сравнения NaN с другими значениями всегда не равны (включая его собственное). Поэтому нельзя сравнивать с , Number.NaNчтобы определить, что значение не является числом, а можно только вызвать isNaN()для сравнения.

NEGATIVE_INFINITY: Значение представляет собой отрицательную бесконечность. Использование: Number.NEGATIVE_INFINITY. Когда число генерируется в арифметической операции или функции и оно меньше, чем -Number.MAX_VALUE, возвращает это значение. Его можно упростить и заменить на -Infinity.

POSITIVE_INFINITY: Значение представляет положительную бесконечность. Использование: Number.POSITIVE_INFINITY. Когда число генерируется в арифметической операции или функции и оно больше, чем Number.MAX_VALUE, возвращает это значение. Его можно упростить и заменить на Infinity.

Задача
Кодирование в функции whatNumberIsIt. Функция принимает 1 параметр: n. это число.

Чтобы оценить число n. Если n является одной из пяти приведенных выше констант, вернуть одну из этих строк:

"Input number is Number.MAX_VALUE"
"Input number is Number.MIN_VALUE"
"Input number is Number.NaN"
"Input number is Number.NEGATIVE_INFINITY"
"Input number is Number.POSITIVE_INFINITY"
Другие значения должны возвращать "Input number is xxx". xxx означает это число.

Например:

whatNumberIsIt(1/0) should return "Input number is Number.POSITIVE_INFINITY"
whatNumberIsIt(100) should return "Input number is 100"
Вам нужно подумать о том, как правильно и эффективно оценить это, и не забыть isNaN().
