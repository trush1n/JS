Описание:

В JavaScript ваш код выполняется в функции. Давайте шаг за шагом выполним вашу первую функцию JS.
Посмотрите на этот пример:


--------keyword "function"
| ----your function name
| | ---if needed, some arguments will appear in parentheses
| | |
function myfunc(){ ---------your function code will starting with "{"
//you should type your code here
}----------------------------ending with "}"

Если мы хотим вывести что-то на экран, возможно, мы можем использовать Document.write()веб или использовать alert()pop your message,
но Codewars не поддерживает эти методы, мы должны использовать console.log()в вашей функции.
Посмотрите этот пример:

function printWordToScreen(){
var somewords="This is an example."
console.log(somewords)
}

Если мы запустим эту функцию, то This is an example.на экране будет видно. Как видите, console.log()это полезный метод. Вы будете его часто использовать!

Задача

миссия 1:

используйте ключевое слово functionдля определения вашей функции, имя функции должно быть helloWorld(не забудьте () and {})

миссия 2:

используйте ключевое слово var(или letили const) для определения переменной str, значением которой strдолжна быть строка: "Hello World!"(не забудьте =).

миссия 3:

введите console.log()в следующей строке (не забудьте поставить strв скобки).
