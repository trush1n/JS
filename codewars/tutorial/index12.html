Описание:

В уроке № 11 мы узнали, что forцикл может использоваться для обхода массива.
Если мы хотим обойти объект, мы можем использовать вариант for:
for..in. Это может обойти все ключи объекта.

Пример:


function showObjectKeys(obj){



for (var key in obj){


    console.log(key);

  }

}

function showObjectValues(obj){

  for (var key in obj){

    console.log(obj[key]);

  }

}

var ob={item1:"This",item2:"is",item3:"an",item4:"example"};

console.log("keys of ob:")

showObjectKeys(ob);

console.log("values of ob:")

showObjectValues(ob);

Результаты кода:



keys of ob:

item1

item2

item3

item4

values of ob:

This

is

an

example

for..inтакже может использоваться с массивами. Изменение кода выше:



function showObjectKeys(obj){

  for (var key in obj){

    console.log(key);

  }

}

function showObjectValues(obj){

  for (var key in obj){

    console.log(obj[key]);

  }

}

var arr=["one","two","three"];

console.log("keys of arr:")

showObjectKeys(arr);

console.log("values of arr:")

showObjectValues(arr);

Результаты кода:


keys of arr:

0

1

2

values of arr:

one

two

three
Как вы видите, ключи — это индексы элементов массива.
Важно: при использовании for..inс массивом ключ (индекс) всегда является строкой, а не числом.
В примере выше ключи — это «0», «1» и «2».
Мы не можем видеть кавычки, потому что console.log() их не показывает.

Хотя for..inможет использоваться для обхода массива,
это не рекомендуется, поскольку в некоторых случаях порядок может быть неожиданным.
Поэтому рекомендуется использовать другой вариант для этого:
for..of(новый в ES6). for..ofможет обойти все значения массива (без доступа к ним через индекс).
Функцию showObjectValues()выше можно изменить следующим образом:

function showArrayValues(arr){

  for (var value of arr){

    console.log(value);

  }

}
