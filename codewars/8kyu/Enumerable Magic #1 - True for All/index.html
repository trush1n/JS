Задача
Создайте метод , который принимает два параметра:

последовательность
функция (указатель функции в C)
и возвращает true , если функция в params возвращает true для каждого элемента в последовательности. В противном случае она должна вернуть false. Если последовательность пуста, она должна вернуть true, поскольку технически ничто не провалило тест.

Пример
all((1, 2, 3, 4, 5), greater_than_9) -> false
all((1, 2, 3, 4, 5), less_than_9)    -> True
