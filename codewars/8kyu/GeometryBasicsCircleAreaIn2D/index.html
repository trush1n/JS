Описание:
Эта серия ката познакомит вас с основами решения геометрических задач с помощью компьютеров.

Напишите функцию circleArea/ CircleArea, которая принимает Circleобъект и вычисляет площадь этого круга.
Класс Circleможно увидеть ниже:

// Represents a Circle where center is a Point and radius is a Number
class Circle {
  constructor(center, radius) { 
    this.center = center; 
    this.radius = radius;
  }
}
А сам Pointкласс можно увидеть ниже:

// Represents a Point where x and y are Numbers
class Point {
  constructor (x,y) { 
    this.x = x;
    this.y = y; 
  }
}
