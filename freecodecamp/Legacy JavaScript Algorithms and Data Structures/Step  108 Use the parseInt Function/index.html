Используйте функцию parseInt
Функция parseInt()анализирует строку и возвращает целое число. Вот пример:

const a = parseInt("007");
Вышеуказанная функция преобразует строку 007в целое число 7. Если первый символ в строке не может быть преобразован в число, то она возвращает NaN.

Используйте parseInt()в convertToIntegerфункции, чтобы она преобразовала входную строку strв целое число и вернула его.
