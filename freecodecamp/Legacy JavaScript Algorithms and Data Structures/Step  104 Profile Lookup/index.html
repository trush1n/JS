Поиск профиля
У нас есть массив объектов, представляющих разных людей в наших списках контактов.

Функция lookUpProfile, которая принимает nameсвойство ( prop) в качестве аргументов, уже написана для вас.

Функция должна проверять, nameявляется ли фактическим контактом firstNameи заданное свойство ( prop) является свойством этого контакта.

Если оба значения истинны, то вернуть «значение» этого свойства.

Если nameне соответствует ни одному контакту, то вернуть строку No such contact.

Если propне соответствует ни одному из допустимых свойств найденного совпадающего контакта, nameто возвращается строка No such property.
